# Products Analysis

## Portfolio Overview

KvanBroekhoven operates a diversified portfolio of B2B SaaS products across agriculture, staffing, time management, manufacturing optimization, and executive intelligence. Each product has established market traction and serves distinct customer segments with proven value propositions.

**The strategic opportunity:** Deploy **AI-powered sales agents** for each product, leveraging Bayata's proprietary platform stack to automate lead discovery, outreach, and deal closure at scale.

**Critical architectural principle:** All agents are deeply integrated with the company Bayata's proprietary system. Bayata owns the API stack and can develop custom hooks/endpoints specifically for agent requirements. The MCP server structure (currently in development at Bayata) might be of use for the agents of KvanBroekhoven

## Product 1: Mestloket

### Market Position & Product Overview
- **Stage:** Live, established (marketplace + compliance platform)
- **Target Market:** Dutch and Belgian agricultural sector
- **Product Type:** Digital marketplace + compliance SaaS
- **Primary Customers:** Dairy farms and livestock operations
- **Geographic Focus:** Netherlands (expandable to EU, preferably Belgium)

### Actual Product Features
- **Mestcalculator:** Free tool for farmers to calculate manure surplus per 170 kg N/ha regulations
- **Online Marketplace:** Digital platform connecting farmers (manure sellers) with buyers (arable farmers, fertilizer companies) by region/postcode
- **Digital VDM System:** Automated Vervoersdocument (transport documentation) for logistics partners with 5-year RVO-compliant archiving and GPS verification

## AI Agent Architecture: Unified Framework Across All Products

### Strategic Architecture Overview

All 5 agents share a **common orchestration framework** while operating product-specific workflows. **Critical principle: All integration occurs through Bayata's proprietary system.**

### Core Agent Components (Per Product)

#### 1. **Discovery Module**
- **Input:** The company Bayata prospect database + external data sources (Lusha, Apollo, Hunter)
- **Process:** Identify prospects matching product ICP (firm size, industry, intent signals)
- **Output:** Scored prospect list (0–100 qualification score)
- **Bayata Integration:** Custom API endpoint for lead discovery and prospect enrichment
- **MCP Tool:** `discover_prospects(product_id, filters)` → calls he company Bayata API

#### 2. **Segmentation & Personalization**
- **Input:** Prospect profile (company, role, pain point indicators)
- **Process:** Assign to segment (large/medium/small, by product)
- **Output:** Segment-specific messaging + outreach angle
- **Bayata Integration:** Custom database table for prospect segments + messaging templates
- **MCP Tool:** `get_messaging_templates(segment, message_type)` → returns Bayata templates

#### 3. **Multi-Channel Orchestration**
- **Email:** Personalized subject lines + body copy (A/B testing framework)
- **LinkedIn:** Connection request + personalized message (timing-based)
- **Social Media:** Relevant content engagement (likes, comments, shares)
- **Bayata Integration:** APIs for email scheduling, LinkedIn automation, social posting
- **MCP Tools:** 
  - `send_email(prospect_id, email_content)` → calls the company Bayata email API
  - `linkedin_outreach(prospect_id, message)` → calls the company Bayata LinkedIn connector
  - `social_engagement(prospect_id, content)` → calls the company Bayata social API

#### 4. **Engagement Tracking (Real-time)**
- **Monitors:** Email opens, clicks, replies; LinkedIn message views, connection accepts; social engagement
- **Scores:** Engagement level (0–100 based on interaction history)
- **Triggers:** Next action in sequence (follow-up email, phone call, LinkedIn reminder)
- **The company Bayata Integration:** Real-time webhooks for engagement events
- **MCP Tools:**
  - `get_engagement_history(prospect_id)` → retrieves the company Bayata engagement data
  - **Webhook receiver:** the company Bayata sends real-time engagement events → Claude processes → triggers next action

#### 5. **Deal Management & Pipeline**
- **Tracks:** Prospect → Lead (engaged) → Opportunity (qualified) → Close
- **Dashboard:** Real-time visibility (discovery count, reply rate, conversion rate, pipeline value)
- **Triggers:** My agent alerts when prospect is ready for direct contact
- **The company Bayata Integration:** Custom deal management API + pipeline tracking
- **MCP Tools:**
  - `create_deal(prospect_id, product_id, stage)` → creates deal 
  - `update_deal(deal_id, new_stage, notes)` → updates deal status
  - `get_pipeline(product_id, date_range)` → retrieves pipeline metrics

#### 6. **Analytics & Optimization (Learning Loop)**
- **Metrics:** Reply rate, close rate, CAC, LTV, ROI per agent
- **Learning Loop:** A/B test results feed back into messaging optimization
- **Reporting:** Monthly performance dashboard (per product, per agent)
- **Bayata Integration:** Custom analytics API + A/B testing framework
- **MCP Tools:**
  - `get_performance_metrics(product_id, date_range)` → analytics
  - `get_ab_test_results(test_id)` → winner determination + recommendations
  - `update_messaging_variant(variant_id, performance_data)` → feedback loop
